##  **Nuxt Configuration**

Add **Pinia** to your Nuxt config (`nuxt.config.ts`):

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@pinia/nuxt'],
  app: {
    pageTransition: { name: 'page', mode: 'out-in' },
  },
  devtools: { enabled: true },
})
```
