##  **Nuxt Configuration**

Add **Pinia** to your Nuxt config (`nuxt.config.ts`):
```bash
npm i pinia @pinia/nuxt
```

Pinia configuration: 

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@pinia/nuxt'],
  app: {
    pageTransition: { name: 'page', mode: 'out-in' },
  },
  devtools: { enabled: true },
})
```
